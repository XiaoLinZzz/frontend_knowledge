### 结论

- JavaScript 中每个对象都有一个原型对象。可以通过 `__proto__` 属性来访问到对象的原型对象
- 构造函数的 *prototype* 属性指向一个对象，这个对象是通过构造函数实例化出来的对象的原型对象
- 原型对象的 *constructor* 属性指向其构造函数
